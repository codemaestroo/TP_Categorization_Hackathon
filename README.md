# TP_Categorization_Hackathon
Репозиторий для командного хакатона на категоризацию писем

## Описание хода работы (логика выполнения):
    1. В файле TODO план-распределение задач и требуемых ролей для работы.
    2. Проанализирована база писем из папки /inbox, для выявления текущих и предполагаемых типов ошибок, и для первого врианта разделения массива писем для их категоризации.
