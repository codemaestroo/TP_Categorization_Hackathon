# Архитектура:

## src - исходники писем отдельно от main тестов

### models.py

    class категорий
    class email
    class ClassificationResult
        категория + причина (почему определили именно сюда) - в логи

    class ProcessingStats
        логи статистика (кол-во входа + кол-во ошибок и верных компиляций)

        логи всего обработанно


### parsers.py

    import re - регулярные выражения

        Часто используемые шаблоны

        \d — любая цифра (0-9).
        \D — любой символ, кроме цифры.
        \w — буква, цифра или подчёркивание.
        \W — любой символ, кроме \w.
        \s — пробельный символ.
        \S — любой непробельный символ.
        ^ — начало строки.
        $ — конец строки.
        . — любой символ, кроме новой строки.
        * — 0 или более повторений.
        + — 1 или более повторений.
        {n,m} — от n до m повторений.


    

tests - проведение тестирований работы проекта

mailbox - распаковка писем, категоризация, логи
